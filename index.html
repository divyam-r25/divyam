epistemic-ops Research

EpistemicOps

EpistemicOps is an OpenEnv-style reinforcement learning environment that simulates how AI agents operate in changing production systems. The environment introduces silent API drift, evolving scenarios, and long-horizon memory challenges, requiring agents to detect changes, adapt their decisions, and preserve useful knowledge across multiple eras. It includes a complete environment loop, realistic mock services, structured reward design, evaluation pipelines, and a live Hugging Face demo.
